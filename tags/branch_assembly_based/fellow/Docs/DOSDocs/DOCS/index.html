Fellow Amiga Emulator

This page contains links to all the documentation for the Fellow Amiga Emulator.
Not all documents are HTML, still this page links them all.

Information Package

The main user manual is in English. There are a number of translations to various languages as well, not all cover the lastest version, but should still get you up and running with the most important aspects of the emulator.
 
 
Information and Documentation:
User Manual Covers basic emulation concepts and configuration of the emulator
Readme.txt Release notes for this version of the emulator
Credits.txt Full credits for work done with the emulator
Fellow.FAQ Frequently asked questions and answers
Tech.txt Technical specification with capabilities and limitations of the emulation
Ledboard.txt Build a Ledboard (Requires DOS character set)
