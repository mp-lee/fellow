Vad är detta program tillför ?
Om du vill göra en hård diskdriver fil till Fellow. Då kan du använda 
detta program.

Hur använder jag det ?
Det är inst så svårt att använda det.
Skriv : Fellowhd.exe (namnet på HD filen) (hur stor ska den vara i Mb)

exampel :
I DOS prompten skriv:
FellowHD.exe AmigaHD 50 (enter)
Den skapar en fil som heter  AmigaHD.ADF och  är 50 Mb stor.

System krav.
CPU :  386 eller högre
Opreation System MS-DOS 5.0 eller högre
